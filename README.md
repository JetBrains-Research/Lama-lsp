
# Lama-LSP

- [ ] textDocument/completion
- [x] textdocument/hover
- [x] textDocument/definition
- [x] textDocument/documentHighlight
- [x] textDocument/references
- [ ] textDocument/formatting
- [x] workspace/symbol
